Requirements

Before you being able to use ICode Framework, there are some requirements. you have to get installed first as follow :

  1. Apache Server (required)
  2. PHP 5.3 or later (required)
  3. Database System like MySQL (recommended)

You can get all those packed if you installed a web server like xampp (win) or lampp (linux) or mampp (mac), and there are many others Web Servers softwares

Installation

you can install ICode in simple steps

1. Unzip the package.

2. Upload the Framewor folders and files to your server. Normally the index.php file will be at your root. or move files to your server root if you host it locally

3. Open the application/config/config.php file with a text editor and set your base URL. If you intend to use encryption or sessions, set your encryption key.

4. If you intend to use a database, open the application/config/database.php file with a text editor and set your database settings.
